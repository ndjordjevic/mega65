**c65-specifications** is the MEGA65 team's corrected and augmented version of the original Commodore C65/C64DX system specification. It is a key primary reference for understanding the hardware the MEGA65 evolved from, complementing the historical background in [[commodore-65-wikipedia]] and the definitive MEGA65 register and memory-map documentation in [[MEGA65-mega65-user-guide]].

_All claims below are sourced from ../../raw/github/MEGA65-c65-specifications.md unless otherwise noted._

## What it does

Provides `c65manualupdated.txt` — an updated plain-text version of the Commodore C65 system specification. The original was `c65manualupdated.txt.gz` from zimmers.net. This version incorporates corrections and additions from various sources plus material figured out by the MEGA65 team. Covers: C65 hardware architecture, VIC-III video chip, 4510 CPU, FDC, CIA, SID, memory map, DOS, and BASIC 10.

## Maintenance status

11 stars, 3 forks, no license, last pushed 2020-06-21. Essentially frozen — the specification is a historical artifact. Useful as the primary hardware reference that informed MEGA65 design decisions, particularly for understanding VIC-III register definitions and the 4510/45GS02 CPU lineage.
