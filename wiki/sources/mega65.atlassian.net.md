The official MEGA65 Confluence wiki — the community's collaborative documentation space, administered by gardners, deft, dddaaann and lydon (write access granted via Discord). Anonymous access shows the structure but blocks some content; individual pages fetch fine directly (the Documentation Landing Page and the emulator guide are captured in full inside [[mega65.org]]'s raw file).

_All claims below are sourced from ../../raw/web/mega65.atlassian.net.md unless otherwise noted._

## What it does

Top-level sections: Introduction, MEGA≡VISION (community project-share event), M65Connect, BASIC 65 How-tos, Tutorials / Documents / Utils, Compatible Hardware, System Development, Old wiki contents, and MEGA65 Releases.

## Key features

The Releases section carries the canonical release notes — recently updated pages include "Release 0.97.2 — Bugfix Release for the MEGA65 R6A board revision" (Feb 2026), "Release 0.97 — 10th Anniversary Edition" (Feb 2026), MEGAFLASH Testing, and SD Card Manager. Known high-value pages referenced by other sources in this wiki: "Ethernet tools mega65_ftp and etherload" (see [[wiebow.mega65.com]]), "Using petcat for cross-developing BASIC programs" (see [[dansanderson.com]]), and the emulator quick-start.

## When to use

For release notes, hardware-revision-specific information (R3/R6/R6A), BASIC 65 how-tos, and compatible-hardware lists. Fetch specific pages directly by URL rather than crawling the space — anonymous Confluence rendering is unreliable at the space level.
