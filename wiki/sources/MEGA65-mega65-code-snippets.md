The official meta-repository: a curated link directory of MEGA65 development resources — assemblers, IDEs, toolchains, example repos, complete games, and core-porting frameworks. It contains no code itself; its value is the map of the community's GitHub ecosystem.

_All claims below are sourced from ../../raw/github/MEGA65-mega65-code-snippets.md unless otherwise noted._

## What it does

Single README organizing the ecosystem into: environment setup (ASM/BASIC), toolchains (official and third-party), code snippets and examples, tutorials, complete projects/games, and alternative cores.

## Key features

- **Assemblers/IDEs**: BSA (Bit Shift Assembler, 6502/65C02/45GS02), Eleven (BASIC IDE running on the MEGA65 itself), C64Studio (ACME-syntax MEGA65 assembly + BASIC 10 compiler), S65 (Shallan's Kick Assembler macro toolkit).
- **Toolchains**: official [[MEGA65-mega65-tools]] + M65Connect; third-party m65dbg (serial debugger), Mega65Toolkit (smnjameson), mega65-basic interpreter, MEGA65-SDK and C65 Firmware Build Kit (SvOlli), Millfork support, Freezer V2, NCM converters.
- **Examples/tutorials**: m65-examples (mteufel), m65_example_code (smnjameson), B10SEX (super-extended character mode from BASIC), TRSE NCM SmoothScrolling, and [[RetroCogs-Mega65Tutorials]].
- **Complete projects**: mega65-weeIP (TCP/IP stack), gardners/raycaster (Wolfenstein-style 3D), megafighter, BSZ (Z-machine interpreter), HitBlock, just-pixel (pixel editor), Manche (MOD-player front-end), M3wP Solitaire, dragonrock (RPG), m65wator, megarogue, mega65puyo, RASTER65, Tilemizer (image→tile/map converter).
- **Core-porting**: sy2002's zxuno4mega65, gbc4mega65, QNICE-FPGA, and the MiSTer2MEGA65 framework for porting MiSTer cores.

## When to use

The first stop when looking for "has someone already built X for the MEGA65" — sample code for a hardware feature, a tool, or a game engine to learn from. Contribution model: open an issue with a link.
