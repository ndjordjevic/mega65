## README
An updated version of the Commodore C65/C64DX system specification. The original specification can be found at http://www.zimmers.net/anonftp/pub/cbm/c65/c65manualupdated.txt.gz

The [Commodore 65](https://en.wikipedia.org/wiki/Commodore_65) (C64DX) was intended as the final 8-bit Commodore, successor to the C64. Development was halted at the prototype stage.

This repo contains:
- `c65manualupdated.txt` — the corrected and augmented specification text. Contains corrections and additions from multiple sources, as well as material figured out by the MEGA65 team.
- `README.md` — short description

The updated specification covers: C65 hardware architecture, VIC-III video chip, 4510 CPU, FDC, CIA, SID, memory map, DOS, BASIC 10, and other system components — providing the technical foundation that informed MEGA65 design decisions.
