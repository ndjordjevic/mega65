## README
The cc65 cross-compiler suite

cc65 is a complete cross-development package for 65(C)02 systems, including a powerful macro assembler, a C compiler, linker, archiver, simulator and several other tools. cc65 has C and runtime library support for many of the old 6502 machines, including: Apple II, Atari 400/800/XL/2600/5200/7800/Lynx, Acorn BBC, Commodore C64/C128/C16/Plus-4/VIC-20/CBM/PET, Commander X16, NES, Watara Supervision, and more. A generic configuration to adapt cc65 to new targets is also available.

Originally based on "Small C" by Ron Cain, enhanced by James E. Hendrix; extensively rewritten by Ullrich von Bassewitz. Core team members: Christian Groessler, Greg King, groepaz, Oliver Schmidt and others.

Main documentation at https://cc65.github.io/doc. For questions: mailing list at https://cc65.github.io/mailing-lists.html or GitHub Discussions, IRC #cc65 on libera.chat.

- ld65.sgml — linker

## Top-level structure
- dir doc/ — SGML documentation for all tools and targets
- dir include/ — C header files (standard and platform-specific)
- dir libsrc/ — Platform-specific runtime library sources (per target)
- dir src/ — Compiler, assembler, linker, archiver source code
- dir cfg/ — Linker configuration files for each target platform
- dir samples/ — Example programs for various platforms
- dir targettest/ — Target-specific test programs
- dir test/ — General test suite
- dir util/ — Utility scripts
- dir asminc/ — Assembly include files
- file Makefile — build system entry point
- file Contributing.md — contribution guidelines
