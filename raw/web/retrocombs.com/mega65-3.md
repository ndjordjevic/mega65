## Overview

Highlights ten distinctive MEGA65 capabilities — combining authentic Commodore computing with modern features (mechanical keyboard, FPGA, multiple storage options).

## Key Technical Content

- Hardware: Cherry MX mechanical keyboard, HDMI/VGA, Ethernet, SD card, IEC (IEEE-488), dual DB9 joystick ports, internal floppy, Xilinx Artix-7 A7-200T FPGA
- CPU/memory: 45GS02 (65CE02-derived) at 3.54 MHz, 128 KB expandable to 1 MB, dual SID stereo
- Software: M65 Connect, BASIC 10/Eleven, Manche (MOD player), GEOS port
- Storage/media: `.D81`, `.MOD`/`.SID`, C64 cartridges, SD card, floppy
- Cores: Game Boy, ZX Spectrum (Amiga/Atari planned)
- Companion: MEGAphone (handheld, cellular/mesh networking)
