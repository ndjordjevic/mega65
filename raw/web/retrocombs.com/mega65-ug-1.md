## Overview

Launches retroCombs' chapter-by-chapter series on the official MEGA65 User's Guide v2. Introduces the MEGA65 as a modern FPGA-based realization of the unreleased Commodore 65 — "a living, breathing project" with continuous community updates — and sets up what newcomers need to begin.

## Key Technical Content

- Accessories: Hyperkin Trooper controller, HDMI cables, Ethernet, microSD cards, shielded audio cables
- Display: Wimaxit 8" monitor and other small-screen options
- Input: Tank Mouse variants (tank-mouse.com, retrohax.net mouSTer, USB clone)
- Documentation set: User's Guide, Developer's Guide, BASIC 65 Reference, Chipset Reference, Complete Compendium (all PDFs)
- Community: MEGA65 Discord, Forum; GitHub issue tracker for guide contributions; MEGA65 Confluence Wiki
