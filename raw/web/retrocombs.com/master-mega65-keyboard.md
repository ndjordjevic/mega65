## Overview

Comprehensive guide to the MEGA65 keyboard — layout, command keys, and shortcuts — bridging classic Commodore keys with MEGA65 additions.

## Key Technical Content

- Command keys: RETURN, SHIFT, SHIFT LOCK, CAPS LOCK (40 MHz speedup), ALT, CTRL, MEGA, RUN/STOP, RESTORE, cursor keys
- Shortcuts: CTRL+1…8 colors, text manipulation, tab stops
- Function keys F1–F16: DIR (F3), LIST (F4), monitor (F8), etc.
- Escape sequences: screen modes, scrolling, margins, cursor positioning
- Special: Matrix Mode debugger, Freezer, boot-mode selection
- MEGA palette: 32 indexed colors (0–31), community-named extended colors
- Core selection: NO SCROLL + number key
