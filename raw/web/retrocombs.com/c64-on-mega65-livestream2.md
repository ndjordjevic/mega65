## Overview

Second exploration of C64 games and demos on the MEGA65 onboarding SD card, with load commands and commentary.

## Key Technical Content

- `LOAD "FILENAME"` syntax matching the disk image
- Games: C64Anabalt, Super Natural (Guns 'n' Ghosts), Shotgun, Frogs
- Demos: Dragon II, Surfing, Time for Dole, Triad, Antic, D.Y.C.P., Greenwars, Star Riders, Electric Cafe, others
- 4-player games (Shotgun, Frogs) need a Multiplayer Interface
- `SYS 2064` restarts Shades demo; `NEW` clears program memory
- All run in C64 emulation mode
