## Overview

retroCombs' first MEGA65 article — unboxing and assembling the MEGA65 Developer's Kit, with errata, features, and exploration plans. (The start of the whole series.)

## Key Technical Content

- Hardware: acrylic case, CR1220 RTC battery (not included), FPGA architecture, Ethernet, expandable I/O
- Software: GEOS (SD card), Freezer, firmware upgrades, `.d81` disk swapping
- Performance: ~40× a stock C64
- Setup tasks: load cores, use Freezer, network config, firmware updates
- Case: static-prone acrylic; replacement spacers from PlexiLaser
