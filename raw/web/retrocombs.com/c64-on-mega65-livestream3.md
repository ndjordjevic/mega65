# Part III: Playing the C64 demos from the MEGA65 onboarding SD Card

- URL: https://retrocombs.com/c64-on-mega65-livestream3
- Published: 2022-04-23
- Fetched: 2026-06-11

## Overview

Companion post to a livestream exploring C64 demos on the MEGA65 onboarding SD card, with load commands and descriptions for 13 demos (2007–2021).

## Key Technical Content

- Access via `LOAD "DEMO_NAME"` (spaces shown as `␣`)
- Demos: IRATA (2013), RGBA (2011), 4K PETSCII (2019), The Earth (2018), Prologue (2021), PETSCII TRIP (2019), others
- Hardware: MEGA65 in C64 mode; Hyperkin Ranger/Trooper controllers
- Reference: CSDB (Commodore Scene Database)
- Space bar exits some demos
