## Overview

A portable MEGA65 development setup built from a Windows laptop with C64 keyboard labels, Xemu, and cloud sync to bridge laptop and hardware MEGA65.

## Key Technical Content

- Hardware: Lenovo ThinkPad T480 (i5-8350U, 16GB, 512GB SSD, 14")
- Software: Xemu, VICE, DirMaster, CBM prg Studio
- C64 keyboard label stickers on laptop keys
- File sync: Google Drive folder → Mac mini connected via DSD USB-to-TTL adapter
- Transfer: M65 Connect for `.D81` images to hardware
- Xemu features: `keyboard.cfg`, full-screen, drag-and-drop disk images, screen capture
