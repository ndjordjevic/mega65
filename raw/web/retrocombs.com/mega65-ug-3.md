## Overview

Covers foundational operation: keyboard layout, the screen editor, special features, and switching between native MEGA65 and C64 compatibility modes.

## Key Technical Content

- Keyboard: RETURN, SHIFT, CTRL, RUN/STOP, RESTORE, cursor keys, MEGA key, function keys, ALT, CAPS LOCK
- Character input methods: Shifted Codes, Control Codes, MEGA Codes, Escape Sequences
- Editor: quote mode, insert mode, tab config, windows, 16 extra colors via CTRL+A
- Navigation: CLR/HOME, CTRL+W (word advance), CTRL+I/TAB (tab stops), CTRL+Z (back tab)
- `FONT` command (options A–F) for ASCII/lowercase display
- Freezer menu: save/load states, freeze slots, mount `.D81` images
- `GO64` mode (or cartridge); alternative "C64 for MEGA65" FPGA core
- Disk hardware: SD2IEC, Pi1541, real 1541 drives
