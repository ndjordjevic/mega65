## Overview

Customizing the MEGA65 via its configuration utility, managing the two SD card slots (internal + external), and obtaining bundled software.

## Key Technical Content

- Config access: hold ALT while powering on, press 1 for Configuration Utility
- Pages: Input, Chipset, Video, Audio, Network, Done
- Input: mouse type (Commodore 1351 or Amiga emulation), de-jitter
- Chipset: DMA controller (F018B or F018), boot disk (SD or floppy), time/date
- Video: PAL/NTSC, digital video output, CRT emulation
- Audio: stereo/mono, SID generation (6581 or 8580), channel swap
- Network: MAC address (must randomize if zeros)
- SD: SDHC 4–32GB, two slots, single-card operation
- SD Card Utility: ALT+power → option 2; requires "DELETE EVERYTHING" confirmation
- Bundled software at files.mega65.org (ALL INTRO DISKS – Public package)
