## Overview

Follow-up to the keyboard mastery guide with community corrections: extra shortcuts, ROM booting, BASIC features, function-key customization.

## Key Technical Content

- ROM booting: name files `MEGA65#.ROM` (# = 1–0); hold key during startup to load that ROM
- `SHIFT+RETURN`: next line without parsing the code
- `HELP` key: highlights the first line with a syntax error
- `↑` key: runs `RUN "FILENAME"` with fewer keystrokes
- CAPS LOCK: 40 MHz speed boost in C64 mode
- Exit window mode: RUN/STOP + RESTORE, or HOME twice
- Function keys: `KEY ON`/`KEY OFF`; reprogram with `KEY` + CHR$/PETSCII codes
- `NEW RESTORE`: recover an unsaved BASIC program after a reset
