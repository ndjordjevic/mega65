## Overview

Walks through MEGA65 setup from unboxing to first power-up: connecting peripherals, installing the RTC battery, configuring video/audio, and reaching the BASIC 65 prompt.

## Key Technical Content

- Required: HDMI or VGA display (480p/60Hz NTSC or 576p/50Hz PAL), powered speakers, microSD (4–32GB), CR2032 RTC battery, Philips screwdriver, 9-pin DIN joystick
- Rear connections: 3.5mm audio, microSD, Ethernet, HDMI, VGA, IEC serial bus, cartridge port, power
- Side ports: power switch, Controller Port 1 & 2, reset button
- RTC: install CR2032 **before** first power-up
- Onboarding config keys: F3–F13 adjust time/date; TAB/SPACE select video mode; A audio test; C CRT emulation; RETURN saves
- Four video modes incl. DVI NTSC/PAL and Enhanced (with sound)
- First boot auto-loads the intro disk menu
