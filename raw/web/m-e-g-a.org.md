## MegaSPUTM project page — https://www.m-e-g-a.org/megasputm/

MegaSPUTM is a re-implementation of the SCUMM engine for the MEGA65, created by community developer kibo (GitHub: https://github.com/ki-bo/megasputm). Runs original LucasArts adventure game scripts on MEGA65 hardware using a combination of the engine and original game assets (not included).

Key technical details:
- Leverages MEGA65-specific hardware: multi-layer RRB graphics, digital audio samples, fast loading times, mouse input, real floppy disk compatibility
- Performance reportedly exceeds the Amiga 500 version of Maniac Mansion
- Developed with the Calypsi C compiler (MEGA65-targeted)
- Collaborator M3wP (GitHub: https://github.com/M3wP) working on The Secret of Monkey Island 256-color version

Supported games:
- Maniac Mansion (primary release, enhanced graphics available)
- The Secret of Monkey Island 256-color version (in development as of 2026)
