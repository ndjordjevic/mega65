Announces a short-form BASIC programming competition for the MEGA65's 10th anniversary — entries must fit on a single 80×25 text screen.

## Technical Content

- **Tall Character Mode (TCR)**: enables 8×16 pixel fonts for sharper text display.
- Tools provided for loading custom PC-style fonts onto the system.
