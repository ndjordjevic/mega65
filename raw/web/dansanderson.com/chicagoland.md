## Overview

VCF Midwest recap: Dan displayed his MEGA65 alongside original Commodore 65 prototypes. Community news: mainboard availability, Filehost messaging feature, new C64 core documentation.

## Technical Notes

- C65 prototype CPU runs at 3.54 MHz; 128K RAM, expandable to ~8MB.
- MEGA65 can run software the original C65 prototypes couldn't reliably execute.
