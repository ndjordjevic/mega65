# The MEGA65 Community Survey 2023

- Published: August 14, 2023
- Fetched: 2026-06-11

## Overview

Invites the community to complete an anonymous survey. News: new Galaga arcade core by muse; mainboard R4 skipped in favor of R5 (improved cartridge interface).

## Technical Content

Demonstrates porting a classic C64 BASIC game to the MEGA65's 80-column mode with minimal changes: modify screen memory addresses, and use the `SPEED` command to adjust processor performance for gameplay compatibility.
