# Release Day!

- URL: https://dansanderson.com/mega65/release-day/
- Published: February 25, 2024
- Fetched: 2026-06-11

## Overview

Announces MEGA65 platform release **v0.96** after 14 months of work on the FPGA core, ROM, and system software. Upgrade instructions for 2022 owners, Xemu users, and DevKit owners.

## Technical Highlights

- Ethernet file transfer via updated M65Connect
- New core selection/flashing menu — slot 0 updates without JTAG
- Hardware typing event queue enhancements
- 80×50 text mode option
- MEGA65 cartridge booting support
- ROM adds default sprite images + improved character set handling
- DevKit flashing + Nexys compatibility deferred to v0.97
