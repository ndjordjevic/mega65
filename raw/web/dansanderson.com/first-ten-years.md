## Overview

10-year retrospective of the MEGA65 project: from the original Commodore 65 prototype (1985–1994) to a shipping retail product (2022–2024).

## Timeline

- **1985–1994**: Commodore 65 designed by Bill Gardei; 3-person team; canceled; <200 prototypes survived.
- **2010**: Paul Gardner-Stephen sells his C65 prototype; begins "C65GS" FPGA project.
- **April 2015**: MEGA partners with Gardner-Stephen; relaunch as MEGA65.
- **2015–2016**: keyboard case prototyping, community building.
- **2019**: pre-series units; MEGAPhone research announced.
- **2020**: DevKit (100 units) funded by €69,000 community donations.
- **2021**: pre-orders opened.
- **May 2022**: first batch shipped (400 units).
- **2023**: second batch (400 units).
- **2024**: third batch (800 units); general availability achieved.

## Key Features

- Upgradeable FPGA cores (7 user slots + slot 0 factory safe)
- Multiple ROM versions (original C65, restored, open-source)
- Commodore 64 compatibility mode
- Digital video output (HDMI + VGA)
- Ethernet connectivity
- Intro Disk v4: 264 games and tools

## FPGA Ecosystem

Highly accurate C64, PET, VIC-20 cores; frameworks for alternate architectures within authentic keyboard-wedge form factor.

## Anniversary Contest

"The Great MEGA65 Egg Hunt" — Easter eggs hidden throughout every unit; prize = original 1985 MEGA65 prototype signed by Dave Haynie, RJ Mical, and others (insurance value >$15,000 USD).

## Forward Vision

MEGA65 platform v1.0 targets feature quality and API stability. MEGAPhone research ongoing. Hardware adapts to component availability.
