## Overview

Text adventure games and interactive fiction on the MEGA65: the Z-machine VM, classic Infocom games, and authoring tools.

## Technical Content

- **Ozmoo**: primary Z-machine interpreter for MEGA65; 80-column text display, save-to-disk. Use **Ozmoo Online** to convert Z-machine games into D81 disk images — runs every classic Infocom game.
- **Inform 6/7** and **Dialog**: languages for authoring adventures.
- **PunyInform**: memory-efficient games compatible with MEGA65 and smaller Commodore systems.
- Featured game: *Hibernated 1: Director's Cut*.
- Also shows BASIC techniques for building adventure games.
