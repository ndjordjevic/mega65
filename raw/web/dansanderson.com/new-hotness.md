## v0.96 Key Features to Test

- **Ethernet file transfer** — transfer files to/from MEGA65 over network
- **Hardware accelerated typing** — faster, more accurate keyboard response
- **Enhanced mouse handling**
- New core flashing capabilities (forthcoming)
- Configurable cartridge booting (forthcoming)

## Testing Instructions

1. Back up SD card
2. Use SD Card Utility to prepare fresh test card
3. Transfer core file to SD card root
4. Transfer ROM and system software files
5. Hold specific keys at startup to access core selection menu

## Bug Reporting

Discord `platform-testing` forum or GitHub repos (core, ROM, User's Guide). Confirm whether bugs appear in both RC and previous stable version.
