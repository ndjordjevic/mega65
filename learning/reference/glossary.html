MEGA65 Hardware & FPGA

Glossary

The canonical vocabulary for this learning track. Every lesson uses these words, in these senses.

The big three concepts

FPGA
A field-programmable gate array: a chip full of logic gates whose interconnection is set by a configuration file, so one physical chip can become many different hardware designs.
_Avoid_: "programmable chip", "processor"
Gateware
A hardware design expressed as a file that configures an FPGA — the hardware analogue of software. On the MEGA65, the gateware makes the FPGA be the computer.
_Avoid_: "firmware", "the program"
Core
The specific gateware loaded onto the MEGA65's main FPGA to make it behave as a MEGA65 (or a C64, or another machine). "The MEGA65 core" = the design on the Artix-7.
_Avoid_: "ROM", "OS"
VHDL
The hardware description language the MEGA65 core is written in — the source you will eventually read and write. (Verilog is the other major HDL.)

The chip and its toolchain (current R6 board)

Artix-7
The Xilinx FPGA at the center of the board — the only FPGA on the current (R6) machine. It runs the core and is the computer; all MEGA65 gateware targets it.
Vivado
The Xilinx toolchain that compiles the core's VHDL into the configuration file loaded onto the Artix-7.
R6
The current production board revision, shipping since 2024. In this workspace, every hardware statement refers to R6.
Keyboard
The key matrix is read through six plain I2C IO-expander chips — fixed-function silicon, no programmable logic.

The custom devices the core implements

45GS02
The MEGA65's CPU: an extended member of the 6502 processor family, far faster than a C64's.
_Avoid_: "the 6502", "the processor chip"
VIC-IV
The video chip implemented in the core; produces the VGA and HDMI picture.
SID
The Commodore sound chip; the MEGA65 provides four of them. Stands for Sound Interface Device.
45E100
The fast Ethernet controller implemented in the core.

Board, memory & ports

C65
The unreleased 1991 Commodore 65 the MEGA65 recreates and extends — the reason the chipset exists.
Attic RAM
The extra memory beyond the classic C64/C65 RAM: a 64 MiB SDRAM on the R6 board.
_Avoid_: "the RAM" (ambiguous — say Attic RAM or chip RAM)
IEC
The Commodore serial bus port for disk drives and printers.
Cartridge port
The C64-style expansion slot on the side; exposes the system bus to plug-in hardware.
DE-9
The 9-pin port (two of them) for joysticks, paddles, and mice. Often miscalled "DB-9".
_Avoid_: "DB-9"
